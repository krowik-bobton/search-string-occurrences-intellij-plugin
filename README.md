# Search String Occurrences (IntelliJ Plugin)

This repository contains the source code for a Kotlin coroutine-based
file search engine used in an IntelliJ IDEA plugin project.

## Features
- Searches for a given string recursively through files in a directory
- Avoids virtual filesystems (`/proc`, `/sys`, `/dev`, `/run`)
- Streams results as a Kotlin Flow of occurrences
